# Sample Practices

A portfolio of demo projects and practice implementations.

## Projects

- **YFL Form** - Youth & Family Life registration form with multi-step validation

## Tech Stack

- React + TypeScript
- Vite
- TailwindCSS
- React Router
- Framer Motion
- React Hook Form

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

This project is configured for deployment on Vercel.
